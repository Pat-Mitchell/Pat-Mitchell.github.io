Textured Mesh

This demo creates a mesh using webGL shaders and maps it onto the mesh.

The texture coordinates are taken directly from the obj files used for the mesh and then they are fed into webGL to be used in the shaders.

The texture, itself, is derived from a tutorial I was following about using glsl shaders to create textures from noise. I reused the code because I didn't have an image that properly wrapped the meshes I have saved and I like how this turned out.
